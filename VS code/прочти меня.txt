Запускаете прогу
заходите в терминал 
вводите: curl -X POST -H "Content-Type: application/json" -d "{\"original_url\": \"https://www.youtube.com/watch?v=dQw4w9WgXcQ\"}" http://localhost:8080/shorten

получаете сокращённую ссылку, которую можно вставить в любой поисковик(по крайней мере с яндексом всё работало)

так же есть счётчик, который считывает кол-во переходов по сокр ссылке(сбрасывается после закрытия проги)
